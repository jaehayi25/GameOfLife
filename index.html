Conway's Game of Life

Jaeha Yi

This is my first program using HTML, JavaScript, and CSS! I implemented Conway's Game of Life, a self-replicating system created by John Horton Conway in 1970.

Rules:

  1. Click on the tiles to color cells black (alive) or white (dead).
  2. Dead cells become alive if there are exactly three neighboring alive cells.
  3. Alive cells become dead if there is < 2 or > 3 neighboring alive cells.
  4. Cells outside of the grid's boundaries are dead.

Although its rules are simple, if there is sufficient space and memory, the program can execute any computer algorithm.
